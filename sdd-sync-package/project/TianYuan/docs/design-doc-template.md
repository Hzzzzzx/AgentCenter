# 设计文档模板

> 被 AGENTS.md 引用的设计文档目录结构及模板规范。

## 目录结构

每个功能模块一个设计文档，放在 `docs/{module}/` 目录下：

```
docs/
├── editor/
│   ├── git-blame-gutter-design.md    # Git Blame 功能设计
│   └── monaco-integration-design.md  # Monaco 编辑器集成设计
├── sidebar/
│   └── explorer-design.md            # 文件浏览器设计
└── ...
```

## 文档结构模板

```markdown
# {功能名} 设计

> 状态：设计中 / 已实现 / 已归档

## 概述
（一段话描述功能）

## 交互设计
（用户如何使用这个功能）

## 技术实现
（架构、关键文件、核心逻辑）

## 防御性编程
（发现的坑、null guard、特殊处理）

## 已知限制
（当前实现的限制）

## 设计决策记录

| 决策点 | 选项 | 最终选择 | 原因 |
|--------|------|----------|------|
| ... | A) ... / B) ... | A | ... |
```
