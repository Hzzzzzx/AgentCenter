事件到 UI 的目标映射

OpenCode /event
  message.part.updated
    text
      -> 主回答段落
    reasoning
      -> 思考摘要块
      -> 非公开或过长内容默认折叠
    tool
      pending/running
        -> 工具组呼吸灯
      completed/error
        -> 工具组条目详情
    file / patch
      -> 产物卡片
    step-start / step-finish
      -> 只更新阶段状态
      -> 不单独显示
    snapshot / compaction
      -> 调试信息
      -> 可选小提示
    retry
      -> 当前阶段内的重试状态
    subtask / agent
      -> 子任务段落

  permission.asked
    -> 输入区交互卡片
    -> 活动区只显示“等待授权”

  permission.replied
    -> 更新交互状态
    -> 不显示为 Agent 输出

  question.asked
    -> 输入区交互卡片

  session.status busy
    -> 顶部呼吸状态

  session.idle
    -> 阶段收束

  session.error
    -> 可恢复异常块
